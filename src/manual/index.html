P-CHESS

P-CHESS is a prechess game collection. Traditional chess is amusing and fascinating game. However, some people thinks that chess is too hard to learn and to play. In order to help chess novices in starting chess rules and strategies, instructors have been developed prechess games, simplified games based on chess, focusing some chess feature. Here it is presented some prechess games to you to play against computer or a friend.

Common tasks are explained below:

  • Starting a game
  • Configuring game appearance

P-CHESS includes following games:

  • King Duel
  • Killer Queen
  • Killer Rook
  • Killer Knight
  • Bishops Against Pawns
  • Knights Against Pawns
  • Pawn Battle
